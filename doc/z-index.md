# Z-index styleguide

## Widget

| Widget   | z-index      |
| -------- | ------------ |
| text     | z-index: 110 |
| table    | z-index: 120 |
| chart    | z-index: 130 |
| button   | z-index: 140 |
| checkbox | z-index: 150 |
| filter   | z-index: 900 |
